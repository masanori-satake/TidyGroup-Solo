# TidyGroup-Solo 技術仕様書

## グループ状態の分類ロジック
- **Active (実況中)**: `localGroupId` が存在。現在のウィンドウにタブとして展開されている。
- **Stashed (冬眠中)**: 保存されているが、現在はどのウィンドウでも開かれていない。
- **Mixed (重複混在)**: 同じ `title` を持つ保存済みグループが複数存在する。
- **Stale (放置)**: 「Stashed」状態かつ、30日以上更新がないグループ。

## 主要アクションの定義
- **Smart Merge (スマート・マージ)**:
  1. 重複した保存済みグループを特定。
  2. すべての重複エントリからURLを抽出。
  3. 最新の「Active」グループをマージされたURLリストで更新。
  4. 重複していた古い「Saved」エントリを削除。
- **Batch Cleanup (一括クリーンアップ)**:
  - 「Stale（放置）」または「Empty（空）」の条件に合致する「Stashed」グループを削除。
