# TidyGroup-Solo 技術仕様書

## タブグループ状態の分類ロジック

- **Active (アクティブ)**: `localGroupId` が存在。現在のウィンドウにタブとして展開されている。
- **Stashed (休止中)**: 保存されているが、現在はどのウィンドウでも開かれていない。
- **Mixed (重複混在)**: 同じタブグループ名のタブグループが複数存在する。
- **Stale (放置)**: 休止状態かつ、所定期間(例:30日間)以上更新がないタブグループ。

## 主要アクションの定義

- **Smart Merge (スマート・マージ)**:
  1. 重複した保存済みタブグループを特定。
  2. 重複しているすべてのタブグループ内の各タブからURLを抽出。
  3. 最新の「アクティブな」タブグループを、全ステップで抽出されマージされたURLリストで更新。
  4. 重複していた古いタブグループを削除。
- **Batch Cleanup (一括クリーンアップ)**:
  - 「Stale（放置）」または「Empty（空）」の条件に合致する「Stashed」(休止中)タブグループを削除。
